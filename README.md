# USOS

Ultra-Small Operating System (USOS) - This is truly one of the smallest operating systems. Inspired by VSCS, USOS and VSCS are not the same thing. USOS has a full, user-friendly interface (command line), which is significantly more intuitive and convenient than VSCS. Although the USOS 1.0.0 disk image is the same size as VSCS 1.0.1, it offers many more features.

## Functional

1. A full-fledged, self-sufficient bootloader.
2. Convenient command line (for more details, see "Releases").
3. Memory management directly through sectors.
4. Support for loading sectors from different disks.
5. View the contents of the sector.
